# SwellSense Design System v1.0

## Visual Vibe
- **Core Aesthetic:** Dark, high-contrast, professional oceanography tool. 
- **Key Concepts:** Liquid motion, precision data, glassmorphism, readability.

## Color Tokens
- **Surface-Base:** #0A192F (Deep Navy)
- **Surface-Card:** rgba(16, 33, 62, 0.7) with 10px backdrop-blur
- **Brand-Primary:** #00F5FF (Neon Cyan - for "Good" conditions)
- **Brand-Secondary:** #7D26CD (Deep Purple - for "Low" conditions)
- **Text-Primary:** #FFFFFF (Pure White)
- **Text-Secondary:** #94A3B8 (Cool Gray)

## Typography
- **Headings:** 'Inter', Sans-serif, Bold (Tracking: -0.02em)
- **Data/Monospace:** 'JetBrains Mono' (For buoy readings and coordinates)

## Components
- **Cards:** 12px border-radius, 1px border (rgba(255, 255, 255, 0.1)).
- **Buttons:** 8px radius, heavy glow (#00F5FF) on hover.
- **Charts:** Use smooth gradients (Cyan to Transparent) for area fills.

## Spacing Logic
- **Base Unit:** 4px (All margins/padding must be multiples of 4).
